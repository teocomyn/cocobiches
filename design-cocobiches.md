# COCOBICHES · Design System

**Version 1.0 · Avril 2026**

Ce document est la référence unique pour toute production visuelle aux couleurs Cocobiches : print, digital, signalétique, réseaux sociaux, merchandising. Il s'applique aux trois établissements du groupe (Hôtel d'Angleterre, Hôtel du Jeu de Paume, Apparts de l'Oncle Louis) sous la marque ombrelle Cocobiches.

---

## 1. Brand Overview

### 1.1 Positionnement

Cocobiches est un groupe hôtelier indépendant à taille humaine, implanté au cœur de Versailles, à deux minutes du Château. Trois adresses complémentaires sous une même marque, qui partagent une promesse commune : une hospitalité lifestyle, chaleureuse, authentique et engagée.

### 1.2 Valeurs de marque

- **Famille** : un groupe indépendant, sans actionnaire extérieur, porté par des propriétaires-exploitants
- **Authenticité** : une identité forte, incarnée par de vraies équipes
- **Lifestyle** : une expérience d'hébergement qui va au-delà de la simple prestation
- **Qualité** : service soigné, cuisine maison, détail à chaque étape
- **Local** : un ancrage fort à Versailles, des partenaires de territoire
- **Responsabilité** : une démarche RSE active et mesurable

### 1.3 Tonalité générale

Chaleureuse, accessible, soignée. Ni corporate froid, ni familier. Toujours en "nous" collectif. Le client est accueilli comme à la maison.

## 2. Logo System

### 2.1 Logo principal (Main Logo)

Le logo Cocobiches combine :
- **Le symbole** : quatre cercles disposés en carré surmontant un rectangle horizontal (évocation du Braille et d'une signature typographique unique)
- **Le wordmark** "CO / CO / BICHES" en typographie Montecatini Pro

Le logo est conçu pour s'appliquer aussi bien en couleur qu'en matière : embossage, gaufrage, gravure, piqûre, impression, pattern répété. Tout usage décoratif est encouragé, **à condition de ne jamais le déformer ni l'abîmer**.

### 2.2 Symbole seul (Main Symbol)

Le symbole (quatre cercles + rectangle) peut être utilisé seul lorsque la marque est déjà identifiée dans le contexte. Par exemple : favicon, marquage objet, repère dans un document existant, signalétique secondaire.

**Règle d'usage** : sur les réseaux sociaux, le symbole seul est utilisé comme avatar, systématiquement associé à une dénomination de compte lisible qui permet le référencement. Le symbole joue uniquement le rôle de repère visuel.

### 2.3 Logos avec endossement (Destination Lock-ups)

Chaque établissement dispose d'un verrouillage logo + nom d'hôtel, dans la typographie Montecatini Pro :

- **COCOBICHES · Hôtel d'Angleterre**
- **COCOBICHES · Hôtel du Jeu de Paume**
- **COCOBICHES · Apparts de l'Oncle Louis**

Ces lock-ups sont utilisés pour toutes les communications propres à un établissement (site web, carte de visite, signalétique intérieure, papier à en-tête, facturation).

### 2.4 Zone de protection

Chaque version du logo doit être entourée d'une zone de protection minimale égale à la hauteur d'un cercle du symbole. Aucun élément graphique, image ou texte ne peut empiéter sur cette zone.

### 2.5 Tailles minimales

- Digital : 32 px de hauteur minimum pour le logo complet, 16 px pour le symbole seul
- Print : 10 mm de hauteur minimum pour le logo complet, 5 mm pour le symbole seul

### 2.6 Interdictions

- Ne pas déformer les proportions
- Ne pas modifier les couleurs hors palette officielle
- Ne pas ajouter d'ombres, de dégradés ou de contours
- Ne pas placer le logo sur un fond insuffisamment contrasté
- Ne pas faire tourner le logo

## 3. Color Palette

### 3.1 Couleurs primaires

Trois couleurs composent l'identité Cocobiches. Chacune a son usage, son rôle et sa signification.

#### Bleu Marine Cocobiches (Primary)

La couleur signature. Utilisée pour le logo, les fonds institutionnels, la typographie titre.

```
HEX    : #2D3077
RGB    : 45, 48, 119
CMJN   : C 100 / M 95 / Y 20 / N 0
Pantone: 2747 C (équivalence)
Usage  : logo, titres, fonds premium, CTA, signalétique principale
```

#### Crème Cocobiches (Secondary)

La couleur chaleur et douceur. Utilisée pour les fonds clairs, les contrepoints, les éléments décoratifs.

```
HEX    : #F4E7DB
RGB    : 244, 231, 219
CMJN   : C 5 / M 10 / Y 15 / N 0
Pantone: 7527 C (équivalence)
Usage  : fonds clairs, textures, backgrounds, typographie sur fond marine
```

#### Vert Cocobiches (Accent)

La couleur vitalité et fraîcheur. À utiliser avec parcimonie comme accent, notamment sur les éléments décoratifs, la nature, le jardin.

```
HEX    : #54A16B
RGB    : 84, 161, 107
CMJN   : C 70 / M 15 / Y 70 / N 0
Pantone: 7739 C (équivalence)
Usage  : accents, éléments décoratifs, signalétique "chambre prête", touches de nature
```

### 3.2 Couleurs fonctionnelles

Pour les interfaces digitales et les documents, voici les couleurs utilitaires complémentaires :

```
Noir texte     : #1A1A1A
Gris foncé     : #4A4A4A   (texte secondaire)
Gris moyen     : #9B9B9B   (texte désactivé, bordures)
Gris clair     : #E5E5E5   (séparateurs, arrière-plans neutres)
Blanc pur      : #FFFFFF   (fonds, espaces négatifs)
Rouge erreur   : #D64545   (messages d'erreur, uniquement interface)
```

### 3.3 Règles d'association des couleurs

**Combinaisons recommandées :**
- Bleu marine sur crème (combinaison signature)
- Crème sur bleu marine (inversion signature)
- Vert sur crème (accent nature)
- Bleu marine sur blanc (usage digital)

**Combinaisons interdites :**
- Vert sur bleu marine (contraste insuffisant)
- Crème sur blanc (contraste insuffisant)
- Vert et crème sans bleu marine en structurant

### 3.4 Ratios d'utilisation recommandés

Sur un visuel type, la répartition idéale est :
- **60% Bleu marine** (structure, identité forte)
- **30% Crème** (respiration, fonds)
- **10% Vert** (accents, touches de vie)

## 4. Typography

### 4.1 Typographie principale : Montserrat

**Usage** : tous les supports courants · site web, affichages, infos pratiques, communications digitales, documents internes, réseaux sociaux, signalétique générale, contenus éditoriaux.

Famille : Montserrat (Google Fonts, open source)
Poids disponibles : Light, Regular, Medium, SemiBold, Bold, Black
Langues supportées : alphabet latin étendu (français, anglais, allemand, portugais, espagnol, italien).

**Hiérarchie recommandée :**

```
H1 · Montserrat Bold    · 48 / 56 px   (web) ou 36 pt (print)
H2 · Montserrat Bold    · 36 / 44 px   (web) ou 28 pt (print)
H3 · Montserrat SemiBold · 24 / 32 px  (web) ou 20 pt (print)
H4 · Montserrat SemiBold · 20 / 28 px  (web) ou 16 pt (print)
Body Large · Montserrat Regular · 18 / 28 px
Body · Montserrat Regular · 16 / 24 px
Caption · Montserrat Medium · 14 / 20 px
Footnote · Montserrat Regular · 12 / 16 px
```

### 4.2 Typographie de marque : Montecatini Pro

**Usage** : typographie sacrée, réservée strictement à trois cas précis :
1. Le wordmark du logo (CO / CO / BICHES)
2. Les endossements de destination (HOTEL D'ANGLETERRE, HOTEL DU JEU DE PAUME, APPARTS DE L'ONCLE LOUIS)
3. La signalétique des numéros de chambres

Cette typographie ne doit **jamais** être utilisée pour du corps de texte, des titres éditoriaux, des réseaux sociaux ou toute autre communication courante. C'est la typographie patrimoniale de la marque, son usage doit rester exceptionnel.

### 4.3 Typographie de titrage : Montserrat Bold distordue

Pour créer un impact visuel fort sur certains supports (accueil, signalétique Bienvenue, vitrophanie, cartes de bienvenue), la marque utilise un traitement spécifique de **Montserrat Bold appliqué à un filet distordu**.

Exemples d'usage :
- "BIENVENUE / WELCOME / BEM-VINDO / WILLKOMMEN"
- "HELLO"
- Titres événementiels saisonniers

**Règle** : la distorsion doit rester lisible, toujours légèrement penchée vers la droite, jamais caricaturale. Le rendu doit évoquer un filet textile souple, pas un effet "word art".

### 4.4 Règles typographiques générales

- Toujours privilégier la lisibilité sur l'effet
- Ne pas mélanger plus de 2 poids dans un même bloc
- Interligne minimum : 1.4 pour le corps, 1.2 pour les titres
- Éviter le justifié sur les petits blocs de texte
- Pas de souligné (réservé aux hyperliens web)
- Pas d'italique sur Montecatini Pro

## 5. Graphic Elements

### 5.1 Patterns

Deux patterns sont déclinables sur tous les supports physiques et digitaux : impressions, textiles, emballages, objets, signalétique.

**Pattern #1 : Dots & Bars**

Reprise géométrique du symbole du logo. Alternance de colonnes de cercles et de rectangles, dans un rythme régulier. Disponible en :
- Bleu marine sur crème (combinaison signature)
- Crème sur bleu marine (version inversée)

**Pattern #2 : Logo repeat**

Répétition du logo complet à 45 degrés, dans un rythme serré. Disponible en :
- Crème sur bleu marine (premium)
- Crème sur crème ton sur ton (élégant, pour textiles haut de gamme)

**Usages recommandés :**
- Tote bags, coussins, foulards, polos
- Pages de garde de documents, intérieurs de cartes
- Emballage cadeau, papier de soie
- Signalétique décorative grand format

### 5.2 Iconographie

Une bibliothèque d'icônes maison est en cours de constitution. Elle reprend les codes graphiques du logo : formes simples, pleines, arrondies, en bleu marine sur fond clair ou en crème sur fond bleu.

Icônes utilitaires de base :
- Café / Petit-déjeuner
- Wifi
- Prix / Tarif (€)
- Taxe de séjour
- Horaires
- Service (cloche de réception)

**Règle** : toutes les icônes doivent être dessinées au même niveau de détail et avec la même épaisseur de trait. Pas de mix entre icônes pleines et icônes linéaires.

### 5.3 Photographie et iconographie

Le style photographique Cocobiches privilégie :
- **Lumière naturelle** (pas de flash direct)
- **Cadrages vivants** (pas de photos institutionnelles figées)
- **Présence humaine** (équipes, clients avec accord, moments de vie)
- **Détails de matières** (bois, textiles naturels, céramique)
- **Ton global chaud** (jamais de filtres désaturés ou bleuâtres)

À proscrire :
- Photos de stock génériques
- Hôtels vides sans âme
- Plats posés trop "léchés" (préférer le naturel imparfait)
- Images en noir et blanc (la marque vit en couleur)

## 6. Components & Applications

### 6.1 Cartes de visite

Format standard : 85 x 55 mm.

Deux variantes :
- **Recto crème / verso bleu marine** avec logo au centre
- **Recto vert / verso crème** pour les collaborateurs événementiel

Informations à inclure systématiquement : logo établissement, nom / fonction, email, téléphone, adresse.

### 6.2 Papier à en-tête

Format A4. Logo en haut à gauche, bloc contact en pied de page, corps du document en Montserrat Regular 11 pt, interligne 1.5.

### 6.3 Signalétique portes (door hangers)

Quatre déclinaisons, chacune dans une couleur dédiée :

| Message | Couleur de fond | Typographie |
|---|---|---|
| BIENVENUE / WELCOME / BEM-VINDO / WILLKOMMEN | Bleu marine | Montserrat Bold distordue, crème |
| DO NOT DISTURB | Bleu marine | Montserrat Bold, crème |
| ROOM CHECKED (chambre prête) | Vert | Montserrat Bold, crème |
| MAKE UP YOUR ROOM (nettoyage en cours) | Crème | Montserrat Bold, bleu marine |

### 6.4 Keycards (cartes électroniques · Jeu de Paume)

Trois variantes de couleur pour différencier les types de chambres ou simplement l'expérience :
- Crème avec numéro bleu marine (chambres standards)
- Verte avec numéro crème (chambres premium)
- Bleu marine avec numéro crème (suites et familiales)

Format standard carte bancaire. Logo établissement en pied de carte. Icône Wi-Fi en haut.

### 6.5 Porte-clés (Apparts de l'Oncle Louis)

Médaillon rond en métal avec gravure du numéro d'appartement. Lanière en cuir bleu marine avec inscription "APPARTS DE L'ONCLE LOUIS" en typographie crème.

### 6.6 Étiquettes en cuir

Petits patchs carrés en cuir bleu marine avec symbole Cocobiches crème, appliqués sur :
- Tote bags
- Coussins
- Bouteilles de cosmétiques
- Porte-clés textiles

### 6.7 Affichage tarifaire (in-room)

Panneau bleu marine avec hiérarchie claire :
- Logo établissement en tête
- Nom de catégorie de chambre (H3 crème)
- Description courte (Body crème)
- Fourchette tarifaire (H3 crème)
- Icônes de services en pied

### 6.8 Vitrophanie

Deux variantes selon les établissements :
- **Vitrophanie identitaire** : grand logo + nom d'hôtel en crème sur vitre teintée
- **Vitrophanie décorative** : frise de symboles Cocobiches répétés horizontalement

### 6.9 Branding merchandising

Produits textiles et accessoires à la charte Cocobiches :
- Polos équipe en bleu marine avec symbole brodé crème
- Chaussettes bleu marine avec symbole crème
- Bonnets vert ou bleu marine avec étiquette cuir
- Coussins jaune moutarde, terracotta, vert, bleu marine, avec symbole crème
- Papier toilette avec symbole (détail d'attention)

### 6.10 Objets et vaisselle

- Verres colorés soufflés (rose, orange, menthe, bleu pâle) avec symbole Cocobiches gravé crème
- Plateaux bleu marine avec logo mat
- Dessous de verre en liège avec logo gravé
- Pochettes textile crème avec inscriptions "VERSAILLES" ou "BASIC NEEDS"

### 6.11 Mobilier intérieur

Palette mobilier recommandée :
- Tons naturels (bois clair, rotin, lin, laine écrue)
- Touches vives en vert Cocobiches (luminaires, petites tables, tabourets)
- Accents bleu marine sur textiles et objets de décoration
- Éviter le noir et le gris anthracite qui durcissent l'espace

## 8. Voice & Tone

### 8.1 Règles rédactionnelles

- **Toujours en "nous"** (jamais en "je" corporatif, jamais en "on")
- **Vouvoiement** systématique avec les clients
- **Tutoiement** possible en interne (pas dans les communications externes)
- **Phrases courtes**, rythmées, jamais jargonneuses
- **Verbes à l'actif**, concrets

### 8.2 Lexique de marque

Mots à privilégier : maison, équipe, famille, convivialité, chaleureux, authentique, sur-mesure, confidentiel, privilégié, à taille humaine, lifestyle, patrimoine, proximité.

Mots à éviter : luxueux, haut de gamme, prestigieux (trop génériques), moderne (vide de sens), révolutionnaire (prétentieux), offre (préférer "séjour" ou "expérience").

### 8.3 Règles par plateforme

- **LinkedIn** : ton professionnel, "nous" corporate, FR uniquement, taguer les partenaires
- **Instagram** : ton lifestyle, légendes FR + EN, usage naturel des emojis
- **Site web** : ton éditorial, FR + EN, SEO optimisé
- **E-mail client** : ton chaleureux, prénom + nom du client dans l'accroche

### 8.4 Emojis

Usage modéré et cohérent. Emojis recommandés pour Cocobiches :
- 🤍 (cœur blanc, signature du groupe)
- 🌿 (nature, jardin)
- ✨ (moments forts)
- 🍽️ ☕️ 🥐 (gastronomie)
- 🏨 (hôtellerie)

À éviter : emojis trop familiers (😂 🤣 💀), trop religieux, trop politiques.

## 9. Do's & Don'ts

### ✅ Do

- Respecter la zone de protection autour du logo
- Privilégier le duo bleu marine + crème comme combinaison signature
- Laisser respirer les mises en page (marges généreuses)
- Utiliser Montecatini Pro uniquement pour le logo et la signalétique chambres
- Signer tous les posts avec hashtag #Cocobiches
- Taguer systématiquement les partenaires locaux
- Montrer les équipes et les vrais moments de vie
- Photographier en lumière naturelle

### ❌ Don't

- Déformer, recolorier ou ajouter des effets au logo
- Mélanger le bleu marine avec d'autres bleus
- Utiliser Montecatini Pro pour du corps de texte
- Copier les codes visuels des grandes chaînes (Accor, Marriott)
- Sur-utiliser les emojis ou les hashtags (maximum 5 hashtags par post)
- Produire des visuels en noir et blanc
- Utiliser des photos de stock génériques
- Placer le vert en grande surface (c'est un accent, pas une couleur principale)

## 10. Checklist de validation

Avant toute publication ou impression, vérifier :

- [ ] Le logo utilisé est-il la bonne version (main, symbol, endossement) ?
- [ ] La zone de protection est-elle respectée ?
- [ ] Les couleurs sont-elles conformes aux codes HEX / CMJN officiels ?
- [ ] La typographie utilisée est-elle Montserrat (jamais Montecatini Pro hors usages autorisés) ?
- [ ] Le contraste est-il suffisant (accessibilité WCAG AA minimum en digital) ?
- [ ] Les crédits photo sont-ils inclus quand nécessaire ?
- [ ] Le ton "nous" est-il respecté ?
- [ ] Les partenaires cités sont-ils tagués (si RS) ?
- [ ] Le contenu est-il cohérent avec la stratégie éditoriale de la plateforme ?
